 # Full-Stack Web Application
# Project Overview

This project is a dynamic full-stack web application developed using PHP and MySQL. The system supports secure user authentication and structured database-driven operations for managing application data efficiently.

The application demonstrates backend logic implementation, database connectivity, and structured troubleshooting practices.

 # Technologies Used

PHP

MySQL

HTML5

CSS3

XAMPP (Local Development Environment)

Git & GitHub

 # Key Features

Secure user authentication system

Dynamic data retrieval and storage

Structured database integration

Organized backend logic

Error handling and debugging implementation



 # See ‘N’ Say Storymaker Web App

An interactive web application that emulates the classic Mattel® See ‘N’ Say Storymaker toy. Users can select words or phrases from dynamic buttons to create custom stories and view the completed sentence.

# Features

Cycle through arrays of words/phrases for each part of a story

Generate a completed story with one click

Responsive, user-friendly interface

Well-structured HTML, CSS, and JavaScript

Project tracked with GitHub version control
